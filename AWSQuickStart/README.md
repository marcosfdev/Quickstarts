# AWS QuickStart

This folder contains a QuickStart guide for setting up projects on Amazon Web Services.

## Setup Instructions

1. **Install the AWS CLI:**

```bash

pip install awscli
```

# AWSQuickStart

2. Configure the AWS CLI:

```bash
aws configure
```
3. Deploy an Application:

```bash
aws deploy ..
```
